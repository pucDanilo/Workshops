# 31.01 UI Components

This document describes the shared UI components found in `src/components/ui`.

## Button
**Location**: `src/components/ui/Button.tsx`

A flexible button component supporting multiple variants.

### Props
- `variant`: 'primary' | 'secondary' | 'text' | 'danger' (default: 'primary')
- `fullWidth`: boolean (default: false)
- Standard HTML Button props (`onClick`, `disabled`, etc.)

## Input
**Location**: `src/components/ui/Input.tsx`

A wrapper around the native input element that includes a label and error message display.

### Props
- `label`: string (optional)
- `error`: string (optional) - Displays validation error message in red.
- Standard HTML Input props (`type`, `placeholder`, `value`, etc.)
- Supports `ref` forwarding for use with `react-hook-form`.

## Card
**Location**: `src/components/ui/Card.tsx`

A container component that provides a white background, rounded corners, and shadow.

### Props
- `children`: ReactNode
- `className`: string

# 32.01 Layout Components

This document describes the layout components found in `src/layouts`.

## MainLayout
**Location**: `src/layouts/MainLayout.tsx`

The primary layout wrapper for the application. It ensures the content area takes up available space and positions the `BottomNav` at the bottom of the screen.

### Usage
Wrap your page routes or content with `<MainLayout>`.

```tsx
<MainLayout>
  {children}
</MainLayout>
```

## BottomNav
**Location**: `src/layouts/BottomNav.tsx`

A fixed navigation bar designed for mobile usage. It conditionally renders navigation items based on the user's authentication status.

### Features
- **Authenticated**: Shows Workshops, Token, and Logout.
- **Unauthenticated**: Shows Login and Register.
- **Active State**: Highlights the current route icon.

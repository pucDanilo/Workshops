
Esta aplicação adota Domain-Driven Design (DDD), separando a solução em quatro camadas: Domain, Application, Infrastructure e UI Console. Além das operações básicas (criar, listar e concluir tarefas) vamos adicionar:

* Listar tarefas vencidas
* Excluir tarefas
* Atualizar título e data de vencimento
